Arukas

发表于 2017-03-17   |   分类于 教程   |  


arukas.io 一款日本的Docker服务。
目前仍在免费测试期内， 截至到2017年3月31日都会提供无限制的Docker服务。
线路和国内直连的， 稳定。

注册

官网：https://app.arukas.io/
注册地址：https://app.arukas.io/sign_up/
填写邮箱， 用户名， 密码注册。
如果收不到激活邮件， 可以用 github 直接登陆。

搭建方式

本人使用的是 CentOS6 + SS

创建应用：
镜像：jdeathe/centos-ssh:centos-6

内存选 512MB
其中 8989 端口是留给默认ss设置端口的， 22端口是 SSH 用的。

启动应用

启动应用，应用会自动部署，等绿了， 显示Running 就说明成功了。稍安勿躁等待一会。

点击绿色的 watch 按钮， 获取 root 权限的密码。

获取密码


复制 password 的值， 保存好。

创建SSH密钥

使用浏览器访问：https://raw.githubusercontent.com/mitchellh/vagrant/master/keys/vagrant 然后将内存保存到一个文本文件。

使用方便的 Xshell 5 进行 SSH 操作。
填入22端口的地址， 不需要前边的 http://

用户名都是默认的 app-admn'， 然后点浏览。

将保存的那个文本导入。
然后选择导入的密钥， 确定。

连接SSH， 使用 sudo -i 切换到 root 账户下。
密码为获取密码步骤拿到的密码， shift + insert 粘贴如命令行。

安装ss

快捷的方式就是使用一键脚本。
秋水逸冰的一键go版本

安装wget：
yum install wget -y

获取脚本：

wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks-go.sh
chmod +x shadowsocks-go.sh
./shadowsocks-go.sh 2>&1 | tee shadowsocks-go.log

输入好 ss 的密码后直接按三次回车，稍等配置完成。

记住密码，就可以直接关掉SSH工具了。

正确食用

实际上Shadowsocks客户端里配置内容不是 SSH 中显示的 服务器IP和端口。
正确的服务器IP和端口在 arukas 控制台刚才创建的 application中。

如图中箭头所指，正确的服务器ip和端口是 (8989/tcp) 所在的那行。

自动获取

arukas 的IP和端口有时候会发生变动， 每天使用要去控制台查看。太过于麻烦。
顺手写了一个利用 arukas 的 api 接口获取ip和端口， 生成 Shadowsocks 配置文件的工具。

arukas-get

使用方式：
一种是有安装 golang ， 填写 main.go 文件里的 Token 和 Secret 和 ID，go build 编译文件后放在 Shadowsocks 根目录，然后为 arukas-get.exe 创建一个桌面快捷方式。以后直接运行这个程序会获取 ip和端口生成配置文件，自动运行 Shadowsocks 程序。

另一种是下载 releases 的压缩包，解压到 Shadowsocks 根目录
填写 apikey.txt 文件里的 Token 和 Secret 和 ID， 为 arukas-get.exe 创建一个桌面快捷方式。以后直接运行这个程序会获取 ip和端口生成配置文件，自动运行 Shadowsocks 程序。
